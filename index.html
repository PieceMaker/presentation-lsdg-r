History

  • R began as the S programming language
    • Developed at Bell Labs in the mid-1970s by John Chambers
    • Goal was to quickly analyze and visualize project data
    • Interpreted language designed to wrap and/or replace Fortran routines
  • S programming language independently rewritten after breakup of AT&T
    • This rewrite known as the R programming language
    • Development and initial release occurred in the early-1990s
    • Created as an open source project with GPL license
  • R has become popular in the statistics community
  • R has been slowly overtaking proprietary SAS software
  • Commercial/Enterprise support provided by multiple companies
    • Revolution Analytics and RStudio among biggest names
  • R Consortium formed in 2015 to guide future of R language
    • Members include IBM, Microsoft, RStudio, Google, and Oracle

Features

  • Tools for building models and visualizing data
    • Can build linear models, non-linear models, machine learning models, time series models, etc...
    • Various graphics libraries including base plot and added package ggplot2
  • Package management system for sharing code
    • Known as Comprehensive R Archive Network (CRAN)
    • Currently contains more than 12,000 packages
    • Packages cover many topics such as statistics, graphics, finance, economics, physics, etc...
  • IDE created by RStudio (named RStudio of course)
    • Provides code highlighting
    • Provides git integration
    • Provides tools to facilitate package development, including documenting and testing
  • Web framework created by RStudio (not named RStudio but Shiny)
    • Combines web client and server-side javascript engine with R
    • Allows client to display R plots and make calls to R for calculations
    • Packages available to provide CSS and Javascript wrappers for libraries such as Bootstrap and D3.js
  • Integration with Microsoft products
    • Revolution Analytics purchased by Microsoft in 2015
    • R integrated with MSSQL starting in 2016
    • Helping power Cortana Intelligence in Azure
